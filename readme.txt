Locker System lets you create unique locker numbers, manage countries and cities dynamically, require email confirmation, and display a protected listing.

== Description ==
This plugin allows administrators to configure locker creation rules with countries, cities, and optional address fields. 
Users get a unique locker number and must confirm by email before activation.

== Installation ==
1. Upload the `locker-system` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure under **Settings â†’ Locker System**.

== Frequently Asked Questions ==
= Can I add new cities or countries? =
Yes, from the admin settings page.

= Does it support email confirmation? =
Yes, users must confirm via the link sent by email.
